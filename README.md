# AIX Studio - App Inventor Extension Studio

[![Build Status](https://github.com/yourusername/aix-studio/workflows/CI/badge.svg)](https://github.com/yourusername/aix-studio/actions)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](https://github.com/yourusername/aix-studio/blob/main/LICENSE)

A powerful web-based IDE for creating, building, and managing App Inventor Extensions with modern development workflows.

## 🌟 Features

- 🎨 **Web-based IDE** - No installation required, works in browser
- 🚀 **Project Management** - Create and manage multiple extension projects
- ✏️ **Code Editor** - Monaco editor with Java/Kotlin syntax highlighting
- 🏗️ **Build System** - One-click building with Apache Ant
- 🧪 **Testing Framework** - Unit testing with coverage reports
- 📚 **Documentation Generator** - Auto-generate API documentation
- 🔄 **Migration Tools** - Upgrade old projects to modern format
- 📦 **Dependency Management** - Add libraries from Maven Central
- 🎨 **Template System** - Multiple extension templates
- 🔍 **Search Functionality** - Find methods, classes, and dependencies
- 📊 **Analytics** - Project statistics and build metrics
- 🐳 **Docker Support** - Easy deployment with Docker

## 📦 Installation

### Prerequisites

- **Node.js** (version 14 or higher)
- **JDK 11** (Java Development Kit)
- **Apache Ant** (version 1.10 or higher)
- **Docker** (optional, for containerized deployment)

### Quick Start

```bash
# Clone repository
git clone https://github.com/kangrissoft/aix-studio.git
cd aix-studio

# Install dependencies
npm install

# Start development server
npm run dev

# Open http://localhost:3001 in your browser
